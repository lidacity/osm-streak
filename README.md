# osm-streak
TOP User chain of changesets in OpenStreetMap

The list includes those who have been in the chain for more than 365 days.

If the chain ends, it is added to the final list.

Record the start date, end date, and number of days, 

Missed dates: 2024-12-15..17

Attention. The script takes a very long time to run for the first time! Subsequent times are just as long!

The final file **index.html** may be updated periodically
